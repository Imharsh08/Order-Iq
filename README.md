# 🎯 Key Features

## 📌 KPI Cards (8 Metrics)

Real-time management KPIs:

- 💰 Total Revenue  
- 📦 Total Orders  
- 🚚 Shipped Percentage  
- ⏳ Pending Orders  
- 📊 Average Order Value  
- 🧾 Tax Collected  
- 🏭 Active Vendors  
- ⚡ Expedited Orders Rate  

All KPIs dynamically update with filters.

---

## 📈 Interactive Visualizations

Powerful visual analytics:

- 📊 Revenue by Vendor (Bar Chart)  
- 🍩 Order Status Distribution (Doughnut Chart)  
- 🏆 Top Products by Revenue  
- 🚚 Shipping Level Mix  
- 🌍 Orders by City (Top 10)  
- 🥧 Vendor Market Share  

Designed for **executive-level reporting**.

---

## 📅 Smart Period Filters

Switch between:

- Daily  
- Weekly  
- Monthly  
- Quarterly  

All KPIs and charts update instantly.
