# 📊 OrderIQ Management Dashboard

An interactive business intelligence dashboard built to monitor order performance, vendor analytics, and operational insights.

## 🚀 Features

✅ KPI Cards  
- Total Revenue  
- Total Orders  
- Shipped %  
- Pending Orders  
- Average Order Value  
- Tax Collected  
- Active Vendors  
- Expedited Rate  

📈 Visualizations  
- Revenue by Vendor  
- Order Status Distribution  
- Top Products by Revenue  
- Shipping Level Mix  
- Orders by City  
- Vendor Market Share  

📅 Dynamic Filters  
- Daily / Weekly / Monthly / Quarterly

🔍 Drill-Down Features  
- Vendor-level breakdown  
- Product-level deep dive  
- CSV download support

📂 File Upload  
- Supports `.xlsx`, `.xls`, `.csv`
- Uses browser caching (localStorage)

📤 Export Options  
- CSV download  
- Print-ready reports

---

## 🛠 Tech Stack

- HTML5  
- JavaScript  
- Chart.js  
- CSS  
- Excel / CSV Data Handling
